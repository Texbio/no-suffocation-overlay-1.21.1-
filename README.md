## No Suffocation Overlay
#### Minecraft 1.21.1 Neoforge
A small mod that removes the suffocation overlay (when your head is in a block).
</br >-Main reason for this mod was "Just Dire Things" adds a phase armor feature, but doesn't disable the suffocation overlay.
</br >-100% client side
</br >-Made with Claude ai
